# Largest Word in a Sentence (C)

This C program finds the **largest word in a given sentence** by traversing the string character by character and comparing word lengths.

## Problem Statement
Given a sentence, identify and print the word with the maximum length.

## Approach
- Traverse the string using a loop
- Count the length of each word
- Track the maximum word length
- Store the ending index of the largest word
- Print the largest word using index calculation

## Concepts Used
- Strings in C
- Character arrays
- Loops and conditions
- String traversal
- Length comparison logic
